# abusive-level-classification

## Idea:
Multiclass text classification based on TFIDF 

## Models Tested
- Naive Bayes Classifier for Multinomial Models
- Linear Support Vector Machine
- Logistic Regression

## Dataframe columns

- user
- abusive_level
  - Not abusive
  - Ambiguous
  - Mildly abusive
  - Strongly abusive
  - Very strongly abusive
